# pwiii-rafael-calzzetta
Programação Web III por prof João Siles
1. Baixar o Laravel Herd
- Acesse o site oficial: https://herd.laravel.com
- Baixe a versão para seu sistema operacional (Windows ou macOS)
2. Instalar
- Execute o instalador baixado
- Siga o processo padrão (next → next → install)
- Ao finalizar, o Herd já inicia automaticamente
3. Verificar se está funcionando
- O Herd roda em segundo plano (ícone na bandeja do sistema)
Ele já configura automaticamente:
 -PHP
 -Nginx
 -DNS local (.test)
